# Reparto de tareas

CRUD y búsquedas creadas con JavaScript, usando las herramientas Fetch, JQuery y XHR. Estructurando el proyecto con Babel y Eslint. El proyecto se basa en una tienda de Juegos de Mesa llamada (Dadónimos Anónimos), con la capacidad de organizar clientes, creadores, editoriales y juegos.

## Página Web PHP

El index de la página se encuentra en el siguiente archivo:

```
index.html
```
Está situado en el src junto a las demás páginas de que contienen las funcionalidades de las tablas.
